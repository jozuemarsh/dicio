# Dicio | Dicionário Brasileiro Online
No momento apenas em português do Brasil.

## Instalação
Em seu projeto, adicione o Dicio como uma dependência no arquivo `composer.json`:

```
"require": {
    "jozuemarsh/dicio" : "^1"
 },
```

Depois execute:

```
composer install
```

Ou

```
composer require jozuemarsh/dicio
```
